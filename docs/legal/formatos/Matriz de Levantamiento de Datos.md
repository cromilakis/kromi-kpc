## Categorías de Dato Personal comunes:
- Identificación (nombre, RUN, domicilio, teléfono, correo, fecha de nacimiento, nacionalidad, estado civil)
- Académicos / profesionales (títulos, estudios)
- Laborales (cargo, remuneración, fecha ingreso, evaluaciones)
- Financieros (cuenta bancaria, ingresos, deudas, scoring)
- Biométricos (huella dactilar, reconocimiento facial, voz)
- Salud (diagnósticos, tratamientos, medicamentos, exámenes)
- Vida sexual / orientación sexual / identidad de género
- Creencias religiosas / convicciones ideológicas
- Afiliación sindical / gremial / política
- Datos de geolocalización
- Imágenes / grabaciones de video
- Datos de NNA (niños, niñas y adolescentes)
